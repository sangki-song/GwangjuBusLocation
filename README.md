광주광역시 내의 시내버스 위치를 실시간으로 맵에 띄우는 안드로이드 애플리케이션.
소스파일에는 공공데이터포털 API키 및 구글 맵 API키가 제외되어 있습니다.
두개의 위치는 각 AndroidManifest 파일, MainActivity 파일에 있습니다.

사용 API : 
 - 공공데이터포털 TAGO 버스위치정보 (https://www.data.go.kr/data/15098533/openapi.do)
 - 공공데이터포털 TAGO 버스노선정보 (https://www.data.go.kr/data/15098529/openapi.do)
 - Google Maps Platform
